# Testing Commands (`test-*`)

Testing utilities for E2E and component tests.

## Commands

| Command | File | Description |
|---------|------|-------------|
| `/test-e2e` | `test-e2e.md` | E2E test with Page Object |
| `/test-comp` | `test-comp.md` | Component test |

## Examples

```bash
/test-e2e "User login and dashboard access"
/test-comp "ReportFormList filtering"
```

## Shortcut

Use `/cmd-test` for quick access from root.
