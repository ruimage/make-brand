# Review Commands (`rev-*`)

Code quality, review, and optimization tools.

## Commands

| Command | File | Description |
|---------|------|-------------|
| `/rev-code` | `rev-code.md` | Deep code review |
| `/rev-perf` | `rev-perf.md` | Performance optimization |
| `/rev-docs` | `rev-docs.md` | Documentation check |

## Examples

```bash
/rev-code "Check entities/warranty-claims/"
/rev-perf "Analyze ReportFormsList rendering"
/rev-docs "Validate API documentation"
```

## Shortcut

Use `/cmd-review` for quick access from root.
