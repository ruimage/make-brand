# Development Commands (`dev-*`)

General development tools for any project.

## Commands

| Command | File | Description |
|---------|------|-------------|
| `/dev-api` | `dev-api.md` | API endpoint with RTK Query |
| `/dev-skill` | `dev-skill.md` | Create custom skills |

## Examples

```bash
/dev-api "GET /api/users with Zod schema"
/dev-skill "Create TypeScript validation skill"
```

## Shortcut

Use `/cmd-api` for quick access from root.
